# MuLifeSaver

# ❓這是什麽？

這是一個普通的脚本，用於快進一些不能快進的視頻，尤其是香港都會大學的**UNI1002ABW大學核心價值**課程和**UNI1012ABW社會責任課程，**幫助你節省時間，不會將你寶貴的時間浪費在這種沒有意義的課程上。

# 🔧如何使用？

請你在觀看這些視頻時，給自己泡上一杯好茶，拿出你珍藏已久的點心，然後優雅的按下鍵盤上的F12打開開發者工具，點擊console，粘貼注入代碼（第一次使用可能需要在console輸入允許粘貼/allow pasting）即可。視頻並會以倍速播放節省你的時間，當你切換到下個視頻時也不會失去優雅，在console按下Up Arrow Key+ Enter即可繼續播放。

![](https://raw.githubusercontent.com/020darling/CDN/refs/heads/main/img/Screenshot%202026-02-02%20181824.png)

# ❗注意事項

建議最高播放速度為16倍，你可以修改 **`const RATE = 2;`**以調整加速倍數，過高的倍速可能導致注入無法生效，如果在部分小視頻中不生效，你可以調整Javascript context,並選中對應視頻窗口即可，如下圖
